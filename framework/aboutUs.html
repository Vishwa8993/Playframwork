FUTURE AND CONCLUSION:

The web has become the central point of all applications. HTML, CSS and JavaScript technologies have evolved quickly - making it almost impossible for a server-side framework to keep up. The whole web architecture is fast moving towards real-time processing, and the emerging requirements of today’s project profiles mean SQL no longer works as the exclusive datastore technology. At the programming language level, we’ve witnessed some monumental changes with several JVM languages, including Scala, gaining popularity.
That’s why Play 2 was created, a new web framework for a new era.
