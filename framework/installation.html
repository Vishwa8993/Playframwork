Scala Installation :

Step 1 : Verify the JDK installation on your machine. Because Scala needs jdk 1.8 or more.

 

Open the terminal and write Java -version and check JDK version.

If it is 1.8 or more than fine. Else download JDK. For that go to the below link : http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

 

Step 2 : Download Scala Binaries from http://www.scala-lang.org/download/. And install it(which is .msi file).

Step 3: Now you can check Scala on cmd. By just typing Scala.

 

 

 

Install Play Framework for Scala:


Step 1: First check below perquisite installed on your laptop or not:  

1) Java JDK 1.8 or more

2)Scala

- If not installed then follow the step for install Scala.

Step 2: Download and install SBT for Scala.
Download from : http://www.scala-sbt.org/download.html
Step 3: Download and Install Typesafe Activator. Follow the link : http://www.lightbend.com/community/core-tools/activator-and-sbt.

After downloading activator. Unzip it .

Step 4:  Add activator folder path to your environment variable path.
For that : go to the environment variable option of my computer.

 

 

 

Step 5 : Download Play framework starter project from : https://www.playframework.com/download
Step 6 : Extract the downloaded folder to the any directory on computer.
Step 7 : Set the environment variable for above downloaded folder of play framework.
Step 8 : Now play framework is installed in your laptop.
